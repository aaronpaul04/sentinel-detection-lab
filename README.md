\# Sentinel Detection Lab



A collection of custom Microsoft Sentinel analytics rules, hunting queries, and playbooks mapped to MITRE ATT\&CK.



\## Structure

\- `detections/` - Scheduled analytics rules (KQL) organized by ATT\&CK tactic

\- `hunting-queries/` - Threat hunting queries

\- `playbooks/` - Automated response playbooks (Logic Apps)

\- `deployment/` - ARM/Bicep templates for one-click deploy

\- `docs/` - Walkthroughs and screenshots



\## Status

Work in progress. See `docs/` for detection walkthroughs.
